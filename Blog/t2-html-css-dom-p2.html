What is grid based design?
Grid based design is a system for creating order among elements in a website. This is usually represented by a series of intersecting horizontal and vertical lines which serve as guides for element placements.
What is all the hype about responsive webpages?
Responsive webpages are important, especially for the user experience, making websties easy to read and navigate with minimal resizing, panning and scrolling across a wide variety of devices (mobile phones, tablets, desktop computers and laptops)
What is semantic structure?
Semantic structure in web design reinforces the semantics, or meaning, of the information in web pages and applications as opposed to merely defining its visual components.
